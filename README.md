#   -   DriveEase – Car Rental Website
#   Project Overview

-   A responsive frontend car rental website built using HTML, CSS, and JavaScript.

The website includes:

-   modern landing page
-   responsive navigation bar
-   featured cars section
-   "Why Choose Us" section
-   customer testimonials
-   call-to-action (CTA) section
-   footer with navigation and contact information


#   Project Structure

-   `index.html`- main webpage
-   `css/` - stylesheets for the website
-   `js/` - JavaScript for interactive features
-   `images/` - project images and assets
-   `README.md` - project documentation


#   Technologies Used

-   HTML5
-   CSS3
-   JavaScript (ES6)
-   Lucide Icons
-   Google Fonts


#   How It Works

-   index.html loads the complete webpage structure.
-   style.css provides the layout, colors, typography, and responsive design.
-   script.js handles interactive features such as menu functionality and other dynamic behaviors.
-   Users can browse available sections, view featured cars, explore services, read customer reviews, and -    navigate through the website.

#   Features

-   Responsive design for desktop and mobile devices
-   Clean and modern user interface
-   Hero section with clear call-to-action
-   Featured car listings
